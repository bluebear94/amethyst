amethyst (soon to be renamed)

A programming language with both minimalism and utility in mind. It is inspired by TI-Basic.

Example?

$fib = λ
  l = 0
  u = 1
  For i, 2, #1
    t = u
    u += l
    l = t
  EndFor
  u
Endλ
