amethyst (soon to be renamed)

A programming language with both minimalism and utility in mind. It is inspired by TI-Basic.

Example?

_ Returns the nth Fibonacci number.
_ For n = 0, produces an extraneous result of 1.
$fib = λ
  l = 0
  u = 1
  For i, 2, #1
    t = u
    u += l
    l = t
  EndFor
  u
Endλ

Version 0.5.x

Later versions may contain functions missing in earlier versions. The latest is 0.5.11.

Roadmap to 0.6

  • [x] bitwise operators
  • [x] math functions
  • [x] time functions
  • [ ] collection functions
  • [ ] random functions

In order to enhance friendliness to new programmers, the = token will now be interpreted as equality, and assignment will use a left arrow (which will be available for input on the GUI).
